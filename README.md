# 🛒 E-Commerce Project

A full-stack E-Commerce web application built using **Node.js**, **Express.js**, **MongoDB**, and **EJS**. The application provides user authentication, product management, shopping cart functionality, and an admin panel for managing products.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login
* Secure Authentication
* Browse Available Products
* Add Products to Cart
* Remove Products from Cart
* Responsive and User-Friendly Interface

### 🛠️ Admin Features

* Admin Dashboard
* Create New Products
* Edit Existing Products
* Delete Products
* Manage Product Listings

### 📦 Product Management

* Product Image Support
* Product Pricing
* Product Categories
* Product Inventory Display

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/E-Commerce-Project.git
```

### Navigate to Project Directory

```bash
cd E-Commerce-Project
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the root directory:

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Start the Application

```bash
npm start
```

or

```bash
nodemon app.js
```
